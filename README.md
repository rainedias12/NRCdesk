 📌 Descrição do Projeto
Este repositório contém um sistema desktop de Help Desk desenvolvido para o gerenciamento, abertura e controle de chamados de suporte técnico. O projeto foi construído integrando uma interface gráfica robusta a um banco de dados relacional, simulando um ambiente corporativo de atendimento ao cliente e resolução de problemas.

 🚀 Funcionalidades
* **Gerenciamento de Chamados:** Abertura, categorização, atualização de status e fechamento de tickets de suporte.
* **Interface Gráfica (Desktop):** Telas intuitivas desenvolvidas em Windows Forms para navegação fluida do usuário.
* **Integração com Banco de Dados:** Conexão direta e persistência de dados estruturada para armazenar o histórico de atendimentos.
* **Controle de Fluxo:** Tratamento de exceções para garantir a estabilidade da conexão com o banco de dados.

 🛠️ Tecnologias Utilizadas
* **Linguagem Principal:** C# (.NET)
* **Framework Visual:** Windows Forms
* **Banco de Dados:** SQL Server (Modelagem de Dados Relacional)

 🔧 Estrutura do Banco de Dados
O sistema utiliza o SQL Server para armazenar as informações de forma estruturada. O modelo relacional foi desenhado para garantir a integridade dos dados através de chaves primárias e estrangeiras, organizando as tabelas de gerenciamento de chamados.
